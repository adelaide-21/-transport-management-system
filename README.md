# Transport Management System

Система управления транспортными средствами логистической компании

## Описание
Проект демонстрирует использование:
- Абстрактных классов
- Интерфейсов  
- Полиморфизма
- Коллекций Java

## Структура проекта
- \`Vehicle\` - абстрактный базовый класс
- \`Truck\`, \`Bus\`, \`Minivan\` - конкретные реализации
- \`CargoTransport\`, \`PassengerTransport\` - интерфейсы
- \`TransportCompany\` - класс для управления парком

## Запуск
\`\`\`bash
javac TransportManagementSystem.java
java TransportManagementSystem
